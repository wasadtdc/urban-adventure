C++ Learning Tutorial

Welcome to this C++ tutorial website. This website introduces basic C++ programming concepts for beginners.

Hello World

``` 
#include <iostream>

int main()
{
    std::cout << "Hello World";
    return 0;
}
```

This is the first C++ program. It prints "Hello World" on the screen.

Variables

``` 
int age = 18;
double score = 95.5;
char grade = 'A';
```

Variables are used to store data in a program.

If Statement

``` 
if(age >= 18)
{
    std::cout << "Adult";
}
```

The if statement is used to make decisions.

For Loop

``` 
for(int i = 0; i < 5; i++)
{
    std::cout << i;
}
```

Loops are used to repeat tasks multiple times.
