# C++ Tutorial Website

## Project Name

C++ Tutorial Website

## Project Introduction

This project is a simple C++ tutorial website developed as a Vibe Coding assignment. The website introduces several basic C++ programming concepts and provides example code for beginners.

## Main Features

* Introduction to C++
* Hello World example
* Variables and Data Types
* If Statements
* For Loops
* Clean and simple user interface
* Responsive web design

## AI Tools Used

* ChatGPT

## Technology Stack

* HTML5
* CSS3

## How to Run

1. Download the project files.
2. Open `index.html` in any web browser.
3. Browse the tutorial content.
